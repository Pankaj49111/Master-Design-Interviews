## Suggested study path
1. Quick refresh: `Notes` (Cheat sheet, REST/API fundamentals, Git workflows).
2. Design patterns and fundamentals: `LLD Prep` ‚Üí `Design Patterns 2025` and `DesignDataStructures`.
3. Implement and practice: pick 2‚Äì3 LLD exercises (e.g., `DesignATM`, `DesignVendingMachine`, `LLDCarRentalSystem`) and write class models + tests.
4. System-level thinking: `External Resources` ‚Äî read selected chapters (start with `Designing Data-Intensive Applications`, `Designing Distributed Systems`).
5. Interview practice: use `External Resources` archives (ByteByteGo, Grokking, Alex Xu) and `LLD Prep` problems for mock interviews.
